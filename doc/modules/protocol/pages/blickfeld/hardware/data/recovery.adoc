Possible reasons for entering the recovery mode.

|===
| Name | Value | Description

| REASON_UNSPECIFIED ^| 0 | Recovery mode is not active
| REASON_BOOT_ATTEMPTS_EXCEEDED ^| 1 | Recovery mode entered because of too many boot attempts
| REASON_FIRMWARE_UPDATE_FAILED ^| 2 | Recovery mode entered because of a failed firmware update
