= Service: Serial

CAUTION: This API is not yet officially released nor maintained.

This service allows to interface the serial / UART connection. 
 
NOTE: The serial connection is not available for all hardware variants.

[#Stream]
== Stream

Bi-directional stream for UART interface 
 
The communication is terminated with newlines. The baud rate is fixed to 115200.

[#_blickfeld_hardware_services_SerialStreamRequest]
=== Request Stream
