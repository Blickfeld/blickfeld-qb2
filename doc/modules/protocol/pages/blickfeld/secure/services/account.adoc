[#Store]
== Store

Store updated account data. 
 
[NOTE] 
==== 
During account activation this method can be accessed using a JWT token with access level `LEVEL_PUBLIC`. 
It will fail with `UNAUTHENTICATED` if no JWT token is presented. 
====
